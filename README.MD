# 语音手势控制贪吃蛇游戏 (Voice & Gesture Snake Game) web版

一个创新的贪吃蛇游戏，支持语音和手势双重控制，让游戏体验更加有趣和互动。

## 🌟 功能特点

### 1. 多模态控制
- **语音控制** 🎙️
  - 基础方向控制："上、下、左、右"
  - 游戏控制命令："开始"、"暂停"、"继续"、"结束"
  - 支持同义词识别
  - 实时语音反馈

- **手势控制** 👋
  - 手掌方向控制移动
  - 特殊手势控制游戏状态
  - 基于 MediaPipe 的实时手势识别
  - 摄像头预览和状态显示

### 2. 游戏特性
- **多种游戏模式** 🎮
  - 经典模式：基础贪吃蛇玩法
  - 闭眼模式：纯语音反馈
  - 解说模式：AI实时解说
  - 挑战模式：答题吃食物

- **交互反馈** 🔊
  - 实时音效反馈
  - 分数语音播报
  - 状态变化提示
  - 危险预警系统

### 3. 辅助功能
- 游戏设置调节
- 多语言支持
- 自定义命令配置
- 无障碍模式支持

## 🎮 使用说明

### 语音命令列表
| 命令 | 动作 |
|-----|------|
| "向上" | 蛇向上移动 |
| "向下" | 蛇向下移动 |
| "向左" | 蛇向左移动 |
| "向右" | 蛇向右移动 |
| "暂停" | 暂停游戏 |
| "继续" | 继续游戏 |

### 手势命令列表
| 手势 | 动作 |
|-----|------|
| 食指向上 | 向上移动 |
| 食指向下 | 向下移动 |
| 食指向左 | 向左移动 |
| 食指向右 | 向右移动 |
| 手掌握拳 | 暂停/继续 |

## 📝 快速开始

### 环境要求
- Python 3.9+
- 网络摄像头
- 现代浏览器（支持WebRTC）

### 安装步骤

1. 克隆仓库：
```bash
git clone [仓库地址]
cd snakeAI
```

2. 创建并激活Conda环境：
```bash
conda env create -f environment.yml
conda activate snakeai
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

### 运行游戏

1. 启动服务器：
```bash
python src/app.py
```

2. 在浏览器中访问：
```
http://localhost:8080
```

3. 允许浏览器访问摄像头权限
